## ✨ **Key Features**

### 🧑‍🎓 **For Students**
- 🔐 Secure user authentication (email/password)
- 🎥 **Paid & free live classes** – join real‑time sessions
- 📚 **Course marketplace** – browse, filter & enroll
- 📊 **Personal dashboard** – track progress & view enrolled content
- 🎤 **AI pronunciation coach** – real‑time feedback (concept)
- 🔍 **Advanced search & filters** (by category, level, class name)
- 📜 **Certificates** upon course completion

### 👑 **For Administrators**
- Complete admin dashboard
- 📚 **Course management** (CRUD, publish/draft)
- 🎥 **Live class scheduling** (assign to categories, track attendees)
- 🏷️ **Class/category name management**
- 👥 **User management** (add, edit, roles, activate/deactivate)
- 📈 **Analytics dashboard** – revenue, popular categories, user growth
- ✨ **Free live class tool** – add free sessions protected by PIN (`12345678`)

### ⚙️ **Technical Highlights**
- ⚡ Blazing fast with **React 18 + Vite**
- 🔥 **Firebase** – Auth, Firestore (real‑time), Storage
- 🎨 **Tailwind CSS** + **Framer Motion** for modern, responsive UI
- 🌍 **Full Sinhala language support** + English fallback
- 📱 Fully responsive (mobile/desktop)
- 🚀 Deployed on **Cloudflare Pages** (edge‑optimised)
